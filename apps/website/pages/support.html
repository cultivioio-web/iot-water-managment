What's in the Box

  • Sensor Node: Main unit with status LEDs
  • Controller Node: Pump control unit with relay
  • Waterproof Sensor Probe: JSN-SR04T with 2.5m cable
  • USB-C Power Cables: 2√ó 1-meter cables
  • Mounting Hardware: Screws, wall plugs, cable ties
  • Quick Start Guide: Step-by-step installation card

Installation Steps

  1. Mount Sensor Node: Place near the tank, protected from rain. Connect the waterproof probe cable.
  2. Install Sensor Probe: Mount the ultrasonic sensor on top of the tank lid, facing downward into the tank.
  3. Mount Controller Node: Place near your pump's starter. Wire the relay to control the pump.
  4. Power Up: Connect both nodes to USB-C power adapters (5V/2A recommended).
  5. Configure via App: Open the Cultivio app, connect via Bluetooth, and set up your devices.

Normal Operation

  • The sensor node measures water level every 30 seconds
  • Data is sent to the controller via Zigbee mesh
  • Pump automatically starts when water level is LOW (configurable)
  • Pump automatically stops when water level is HIGH (configurable)
  • Check status anytime via the mobile app (Bluetooth connection)

Factory Reset

To reset a device to factory settings:

  1. Press and hold the BOOT button for 10 seconds
  2. LED will blink rapidly, then turn off
  3. Release button ‚Äì device will restart with default settings
  4. Re-configure using the mobile app

‚ùì Frequently Asked Questions

What is the range between sensor and controller?

The Zigbee mesh network provides approximately 30-50 meters line-of-sight range. For multi-story buildings (4+ floors), we recommend adding a Router Node (included in Extended Kit) for reliable coverage.

Does it work without internet?

Yes! AquaSense uses Zigbee mesh networking which works completely offline. You only need internet if you want to access future cloud features (optional subscription).

How do I check status on my phone?

Open the Cultivio web app (cultivio.in/app) on your phone, turn on Bluetooth, and connect to your Controller Node. You'll see real-time water level, pump status, and can control the pump manually.

Can I control the pump manually?

Yes! The app has a manual override feature. You can start the pump for 10, 15, 20, or 30 minutes with a single tap. The system will automatically stop after the selected duration.

What type of tanks does it work with?

AquaSense works with any overhead tank from 200 to 10,000 liters. The ultrasonic sensor can measure depths from 25cm to 450cm. Supports plastic, concrete, and metal tanks.

Is the sensor waterproof?

The sensor probe (JSN-SR04T) is fully waterproof and designed for tank installation. The main sensor node and controller node should be kept in dry, protected locations.

What's the warranty period?

All AquaSense products come with a 6-month warranty against manufacturing defects. Extended warranty (12 months) is available as an add-on during purchase.

How do I update the firmware?

Firmware updates are pushed via the mobile app when you connect to your device. You'll see a notification if an update is available. Updates are optional but recommended for new features and improvements.

üîß Troubleshooting

Device Not Turning On

  • Check USB-C cable connection
  • Try a different power adapter (5V/2A recommended)
  • Check if the power LED lights up

Bluetooth Connection Failed

  • Ensure Bluetooth is enabled on your phone
  • Move closer to the device (within 10 meters)
  • Try closing and reopening the app
  • Use Chrome browser on Android for best compatibility
  • iPhone users: Use Bluefy Browser app

Zigbee Not Connecting

  • Ensure both devices are powered on
  • Check if Zigbee LED is blinking (searching) or solid (connected)
  • Try moving devices closer together initially
  • Perform factory reset and re-pair if needed

Inaccurate Water Level Reading

  • Ensure sensor probe is mounted perpendicular to water surface
  • Check for obstructions in the tank (pipes, floats)
  • Verify tank height is correctly configured in the app
  • Clean the sensor probe face if dirty
